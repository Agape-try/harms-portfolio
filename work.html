Ethiopian Recipe Collection

A comprehensive digital cookbook preserving traditional Ethiopian recipes, cooking techniques, and the cultural stories behind each dish. Features step-by-step instructions, ingredient substitutions, and video tutorials from Ethiopian grandmothers.

Technologies:

Next.js TypeScript Supabase Tailwind CSS

500+ recipes documented, 10K+ users worldwide

Educational Live

Interactive Heritage Map

An interactive map showcasing historical sites, cultural landmarks, and UNESCO World Heritage sites across Ethiopia. Includes 360° virtual tours, historical context, and user-contributed stories.

Technologies:

React Mapbox Node.js MongoDB

200+ locations mapped, 50K+ virtual visits

Educational Live

Amharic Learning Platform

A gamified language learning platform for Amharic, featuring interactive lessons, pronunciation guides, and cultural context. Designed for both children and adults in the diaspora.

Technologies:

Vue.js Python PostgreSQL WebRTC

5K+ active learners, 95% completion rate
